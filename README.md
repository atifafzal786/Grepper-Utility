# Grepper

Grepper is a lightweight, **Tkinter GUI** tool to search folders and files by name, path patterns, and (optionally) file content.

## Quick start

```powershell
python .\grepper.py
```

## Install (optional)

```powershell
python -m pip install "git+https://github.com/<you>/<repo>.git"
grepper
```
